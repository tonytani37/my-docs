style.cssファイル説明

やりたかったこと

  1. 邑南町青少年旅行村の公式HPを元に、もっとかっこよく全面的に見直し。

  2. お知らせは https://microcms.io/ のサービスを利用し、登録されたものをjavascriptで取得して表示するようにしている（全件中最新の２件のみ）

区分 セレクタ / クラス 主な設定内容 表示イメージ（例） 備考
基本文字設定 body font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", "Hiragino Kaku Gothic ProN", Meiryo, sans-serif;
font-size: 16px; line-height: 1.6; color: #333;		 標準テキスト（本文に使われるスタイル） 可読性を重視した標準フォント指定
リンク a / a:hover 通常：color: #1E8449;
ホバー：color: #156236;
text-decoration: none;		 通常リンク → ホバー時 下線なし、ホバーで濃緑に変化
見出し（h2） .section h2 font-size: 1.5rem; color: #1E8449; border-bottom: 2px solid #A08C69; セクションタイトル 見出し＋下線で区切り
見出し（h3） .section h3 font-size: 1.0rem; color: #1E8449; 小見出しタイトル h2より控えめ
ヒーロータイトル（h1） .hero-overlay h1 font-size: clamp(2rem, 5vw, 4rem); color: #fff; text-shadow: 2px 2px 4px rgba(0,0,0,0.5); Hero Title サンプル 背景上でも文字を読みやすく
ヒーロー説明文（p） .hero-overlay p font-size: clamp(1rem, 2vw, 1.5rem); color: #fff; text-shadow: 1px 1px 2px rgba(0,0,0,0.5); Hero サブテキスト 背景映像でも視認性高い
本文 .site-card p font-size: 1rem; color: #666; 落ち着いた本文テキストです。 グレー系で柔らかい印象
お知らせ（リスト） .news-list li font-size: 1.1rem; border-bottom: 1px dashed #eee; 2025-10-23　ニュース項目例 少し大きめ・区切り線あり
強調文字 .phone-number font-size: 2rem; font-weight: bold; color: #A08C69; 012-345-6789 電話番号などの強調表示
ボタン文字 .button text-transform: uppercase; font-weight: bold; font-size: 1.1rem; Click Me 英字は自動的に大文字変換
文字位置 .section text-align: center;
中央揃えのテキスト
 セクション全体の基準
.news-list text-align: left;
左寄せのリスト
 可読性重視
.site-footer text-align: left; font-size: 1.5rem; color: #ccc; フッターテキスト 左寄せ・淡色表示
文字影 .hero-overlay h1, p text-shadow 白文字＋影効果 背景上で視認性を高める
レスポンシブ対応 @media (max-width:500px) .hero-overlay h1 { font-size: 2rem; } 小画面時のタイトルサイズ スマホ用縮小設定
